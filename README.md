# DECA Helper

DECA Helper is a lightweight, fast, and organized study website designed to support DECA students with vocabulary, performance indicators, roleplay examples, and practical tips. The project emphasizes clarity, consistency, and smooth navigation across multiple pages.

---

## Project Structure/
├── index.html              # Homepage
├── vocab.html              # Vocabulary page
├── performance.html        # Performance Indicators page
├── roleplay.html           # Roleplay Examples page
├── tips.html               # Tips & Tricks page
│
├── styles.css              # Global styling for all pages
├── page-transition.js      # Handles page fade/slide transitions
│
├── favicon.png             # Browser tab icon
├── logog.png               # Logo used in the UI
│
└── README.md          




---

## Features

### Multi-Page Layout
Each major DECA topic is separated into its own HTML file for clarity and ease of navigation.

### Unified Styling
All pages share a consistent design through a single global stylesheet.

### Smooth Page Transitions
A dedicated JavaScript file provides subtle transitions between pages for a more polished experience.

### Lightweight and Fast
No frameworks or heavy dependencies. Pure HTML, CSS, and JavaScript.

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla)
